# Adventure Quest

A recreation of the classic Adventure Quest game with a modern web interface.

## Features

- **Login Screen**: Beautiful animated login interface
- **Town Screen**: Explore the town with interactive buildings, NPCs, and UI elements
- **Battle System**: Turn-based combat with spells, attacks, and item management
- **Character Progression**: Level up, gain XP, and manage stats (HP, MP, SP)
- **Persistent State**: Your progress is saved in localStorage

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

## How to Play

1. **Login**: Enter any username and password to start
2. **Explore Town**: Click around the town to see various buildings and NPCs
3. **Battle**: Click the castle to start a battle with a random monster
4. **Combat**:
   - Use **Attack** for basic physical damage
   - Use **Spells** for magical attacks or healing
   - Use **Items** to restore health/mana (coming soon)
   - **Flee** to escape from battle
5. **Win Battles**: Gain XP and gold to level up your character

## Technologies Used

- React 18
- React Router
- Tailwind CSS

Enjoy your adventure!
